# Public / Private Boundary

## Public In This Repository

- Derived CSV tables needed to inspect reported evaluation conclusions.
- Final figure exports in PDF, PNG, and SVG.
- Dataset/source links and redistribution notes.
- Figure and reference audit tables.
- Validation script and checksum manifest.
- License and citation metadata.

## Excluded From This Repository

- Raw third-party data and downloaded archives.
- Model checkpoints and generated prediction masks whose redistribution status is not established.
- Active manuscript source, compiled submission PDFs, cover letters, reviewer-response drafts, and portal files.
- Internal `rounds/`, `logs/`, work notes, gate conversations, and private process records.
- Credentials, API keys, browser/session data, and local machine paths.

## Reproduction Claim

This repository supports public inspection of the derived evidence boundary and figure/data artifacts used by the manuscript. It does not claim to be a complete raw-data regeneration archive.
